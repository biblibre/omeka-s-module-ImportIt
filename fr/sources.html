Sources

Pour pouvoir importer des ressources, vous avez besoin de déclarer une source. Une source a un nom, un type et plusieurs paramètres qui dépendent de son type. Le type de source dicte comment les ressources sont récupérées (si c’est depuis un fichier, une URL, ou autre chose) et transformées avant d’être importées dans Omeka.

Vous pouvez avoir plusieurs sources du même type.

Actuellement il n’y a qu’un seul type de source, METS côté serveur, qui parcoure les fichiers METS locaux et crée les contenus et médias correspondants.

Créer une source

Pour créer une nouvelle source, cliquez sur « Import It » dans le menu de navigation.

Puis cliquez sur le bouton « Ajouter une nouvelle source ».

Donnez un nom à la source et sélectionnez son type, puis cliquez sur le bouton « Ajouter ».

Remplissez les paramètres spécifiques au type, puis cliquez sur le bouton « Enregistrer ».

Pour plus de détails sur les paramètres spécifiques au type, lisez la page correspondante dans Types de source.

Démarrer l’import

Cliquez sur l’icône d’import pour aller sur le formulaire des paramètres d’import.

Une fois le paramétrage fait, cliquez sur le bouton « Démarrer l’import ».
